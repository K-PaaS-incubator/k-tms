일반사항
  1. 하나의 룰은 룰 헤더와 룰 옵션 부분으로 구성됩니다.
  2. 룰 헤더와 룰 옵션은 "(", ")"으로 구분됩니다.
룰헤더
  1. 룰 헤더는 총 7개의 필드로 구성됩니다.
  2. 필드와 필드는 공백 문자로 구분합니다.
  3. 룰 헤더는 "룰타입", "프로토콜", "Src주소", "Src포트","방향", "Dst주소", "Dst포트" 필드로 구성됩니다.
    1. "룰타입"- "alert"을 지정합니다.
    2. "프로토콜"- 다음 4가지 프로토콜을 지정할 수 있습니다.
      1. "tcp"
      2. "udp"
      3. "icmp"
      4. "ip" : "tcp" 또는 "udp" 또는 "icmp"를 의미합니다.
    3. "Src주소", "Dst주소"
      1. 임의 호스트 : "any"
      2. 특정 IP : "x.x.x.x", 1≤x≤255
      3. IP/Netbit : x.x.x.x{/y}?, 1≤x≤255, 1≤y≤32- 예: 127.0.0.1/32, !211.1.1.1/32"!"를 붙여서 exeption IP 주소를 지정할 수 있습니다.
